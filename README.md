# Power BI Portfolio

Welcome to my Power BI portfolio. This repository presents selected analytics projects through business questions, data models, DAX measures, report screenshots, and the decisions each report supports.

This portfolio includes five report case studies spanning sales, marketing, and insurance analytics.

## Important privacy note

Only sanitized material is published here. Tenant URLs, customer or employee information, credentials, connection strings, proprietary data, and confidential business metrics must not be committed. A `.pbix` file should only be included when its data is public or fully synthetic and redistribution is permitted.
